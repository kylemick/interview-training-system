# 科學常識專項說明

## 專項代碼
`science-knowledge`

## 重要性
- **SPCC特別重视**：聖保羅男女中學作为STEM教育重點學校，非常注重學生的科學素養
- 测試學生的好奇心、探究精神和科學思維
- 結合生活实际，考察应用能力

## 考察范围

### 1. 基础科學原理
- 物理：力、热、光、电、磁等基本概念
- 化學：物质变化、日常化學现象
- 生物：人体、動植物、生態係統
- 地球科學：天气、气候、地质现象

### 2. 生活中的科學
- 为什么會有彩虹？
- 飞机为什么能飞起來？
- 食物在冰箱里为什么不容易坏？
- 为什么海水是咸的？

### 3. 环境保护和可持续發展
- 全球暖化及其影响
- 塑料污染問題
- 可再生能源（太阳能、風能等）
- 生物多樣性保护

### 4. 科技發展与創新
- 人工智能和机器學習的应用
- 太空探索（如火星任務）
- 生物科技（如基因编輯）
- 清洁能源技術

### 5. STEM話題
- 编程和計算思維
- 工程设計思維
- 數據分析与統計
- 科學实验方法

## 題目示例

### 基础科學原理類
**Q1**: 解释为什么冬天窗户上會有水珠？这与哪些科學原理有關？

**Q2**: 如果你在月球上跳跃，會比在地球上跳得高吗？为什么？

**Q3**: 为什么植物的叶子大多是绿色的？

### 生活应用類
**Q4**: 为什么医生建議多吃水果和蔬菜？從营養學角度解释。

**Q5**: 为什么飞机起飞時要求關闭电子设備？

**Q6**: 微波炉如何加热食物？为什么不能放金属物品進去？

### 环境保护類
**Q7**: 香港政府推行塑料袋收费，你认为这个措施對环保有帮助吗？为什么？

**Q8**: 如果你是學校的环保大使，你會提出什么建議來减少學校的碳排放？

**Q9**: 全球暖化對香港有什么影响？我们可以做什么？

### 科技發展類
**Q10**: 你认为人工智能會取代老师吗？为什么？

**Q11**: 如果人類可以移民火星，你觉得會面临什么挑战？

**Q12**: 电動车比汽油车更环保吗？從哪些方面考虑？

### STEM综合類
**Q13**: 如果要设計一个机器人帮助老人，你會让它有哪些功能？为什么？

**Q14**: 學校想建一个太阳能發电係統，你觉得应该考虑哪些因素？

**Q15**: 如何用科學方法验证"多喝水對身体好"这个說法？

## SPCC面試特點

### 評分重點
1. **科學思維**：能否用科學原理解释现象
2. **好奇心**：是否對科學問題感兴趣，主動思考
3. **批判性思維**：能否质疑、分析、評估信息
4. **应用能力**：能否将科學知識聯係生活实际
5. **表達能力**：能否清晰地解释科學概念（英文更佳）

### 回答策略
1. **結构化回答**：先說結論，再解释原理，最後举例
2. **展示思考過程**：即使不確定，也要說明自己的推理
3. **聯係实际**：尽量結合生活经验或時事
4. **承认不足**：不知道時诚实說明，表達愿意學習的態度
5. **提出問題**：可以反問面試官，展示好奇心

## AI生成題目的提示要點

当为科學常識類別生成題目時，AI提示詞应包含：
- 适合小學5-6年级學生的知識水平
- 題目要有趣味性，引發思考
- 結合香港本地情境（如香港的气候、环境政策等）
- 鼓励開放性回答，没有唯一標準答案
- 重视思考過程而非死記硬背
- 可以跨學科（如科學+社會+道德）

## 參考答案評分維度

1. **準確性**（30%）：科學概念是否正確
2. **深度**（25%）：是否深入理解原理，而非表面描述
3. **应用**（20%）：能否举出恰当的例子或应用场景
4. **邏輯**（15%）：論述是否有条理，推理是否合理
5. **創新**（10%）：是否有独特见解或創意想法

## 与其他專項的關聯

- **英文口語**：科學常識題常用英文提問（SPCC特色）
- **時事常識**：气候变化、疫情等時事有科學背景
- **邏輯思維**：科學推理需要邏輯思維能力
- **个人成長**：STEM兴趣、科學探索经历

## 訓練建議

1. **日常积累**：關注科普节目、書籍、网站（如NASA、BBC Earth）
2. **動手实验**：在家做简单科學实验，增强理解
3. **提問習惯**：遇到自然现象就想"为什么"
4. **英文學習**：學習科學詞汇的英文表達
5. **跨學科思考**：将科學与社會、道德問題結合

## 常见误区

❌ 背诵科學定义但不理解
❌ 回答太简短，没有展開
❌ 不敢承认不知道，乱猜
❌ 只說理論，不聯係实际
❌ 忽视科學与社會的聯係

✅ 用自己的話解释原理
✅ 举例說明，展示理解
✅ 诚实并展示好奇心
✅ 聯係生活经验
✅ 考虑科學的社會影响
