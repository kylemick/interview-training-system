# Change: 題庫創建使用數據庫學校下拉列表

## Why

目前題庫創建（手工錄入和AI生成）時,學校字段(school_code)需要用户手動输入文本,存在以下問題:
1. 用户可能输入错误的學校代碼,導致數據不一致
2. 无法提供標準化的學校選擇,影响數據质量
3. 用户体验不好,需要記住學校代碼

學校檔案表(school_profiles)已经存储了所有學校信息,应该通過下拉列表的方式供用户選擇,確保數據一致性和用户体验。

## What Changes

- 題庫管理前端页面添加學校下拉列表組件,從數據庫動態加载學校列表
- AI生成題目界面添加學校下拉列表組件
- 手動添加/编輯題目界面的學校字段從文本输入改为下拉選擇
- 確保學校列表实時從`school_profiles`表获取
- 保留空值選項,允许不指定目標學校

## Impact

- 受影响的規格: `question-bank` (題庫管理)
- 受影响的代碼:
  - 前端: `frontend/src/pages/Questions/index.tsx` (題庫管理页面)
  - 後端: 已有 `/api/schools` 接口可直接使用
- 數據庫: 无需修改,已有外键约束 `questions.school_code -> school_profiles.code`
- 用户体验: 提升數據一致性和用户操作便利性
