## 技術栈

- **框架**: React 18 + TypeScript
- **UI庫**: Ant Design 5
- **狀態管理**: Zustand
- **HTTP客户端**: Axios
- **图表庫**: ECharts
- **路由**: React Router v6
- **日期处理**: Day.js
- **构建工具**: Vite
