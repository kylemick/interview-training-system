## MODIFIED Requirements

### Requirement: 練習模式選擇
係統SHALL支持多種練習模式，包括任務模式、自由模式，以及弱點專項練習模式。

#### Scenario: 選擇弱點專項練習模式
- **WHEN** 用户在練習页面選擇"弱點專項練習"模式
- **THEN** 係統显示弱點選擇器
- **AND** 用户可以選擇一个或多个弱點
- **AND** 係統显示每个弱點關聯的學習素材
- **WHEN** 用户選擇弱點和素材并開始練習
- **THEN** 係統基于選定的弱點和素材生成針對性題目
- **AND** 練習過程中显示相關素材提示

#### Scenario: 練習時查看素材
- **WHEN** 用户在練習過程中
- **THEN** 係統在題目旁边显示"查看相關素材"按钮
- **WHEN** 用户點击该按钮
- **THEN** 係統显示相關學習素材的预览
- **AND** 用户可以查看完整素材內容
- **AND** 素材查看次數自動增加

#### Scenario: 反馈結合素材
- **WHEN** 用户完成練習并获得AI反馈
- **THEN** 係統在反馈中結合弱點和相關素材给出改進建議
- **AND** 反馈中引用素材中的關键知識點
- **AND** 提供素材链接供用户深入學習
