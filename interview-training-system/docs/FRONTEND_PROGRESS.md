## 技术栈

- **框架**: React 18 + TypeScript
- **UI库**: Ant Design 5
- **状态管理**: Zustand
- **HTTP客户端**: Axios
- **图表库**: ECharts
- **路由**: React Router v6
- **日期处理**: Day.js
- **构建工具**: Vite
