# Capability: AI Feedback (AI反馈係統)

## ADDED Requirements

### Requirement: 語言质量分析
係統SHALL分析學生回答的語言表達质量并提供改進建議。

#### Scenario: 英文回答的語言分析
- **WHEN** 學生提交英文口語回答
- **THEN** AI評估語法正確性、詞汇丰富度、句式多樣性
- **AND** 给出語言质量評分（0-100）
- **AND** 指出具体語言错误（如"時態错误：用了'goed'应为'went'"）
- **AND** 建議可以使用的高级詞汇或句式

#### Scenario: 中文回答的語言分析
- **WHEN** 學生提交中文表達回答
- **THEN** AI評估用詞準確性、邏輯连贯性、表達流畅度
- **AND** 给出語言质量評分（0-100）
- **AND** 指出表達不清或冗余的部分
- **AND** 建議更简洁或更正式的表達方式

### Requirement: 內容深度評估
係統SHALL評估回答內容的相關性、完整性和深度。

#### Scenario: 評估回答切題程度
- **WHEN** AI分析學生回答內容
- **THEN** 判断回答是否切合問題（切題/偏題/跑題）
- **AND** 如果偏題，指出遗漏的關键點
- **AND** 给出內容相關性評分（0-100）

#### Scenario: 評估回答深度和见地
- **WHEN** AI分析學生回答的深度
- **THEN** 判断回答是否有个人见解或仅表面描述
- **AND** 評估是否提供了支持觀點的例子或論據
- **AND** 给出內容深度評分（0-100）
- **AND** 建議如何深化回答（如"可以补充一个具体例子"）

### Requirement: 弱點識別
係統SHALL識別學生在单次會話和历史記錄中的薄弱环节。

#### Scenario: 单次會話弱點總結
- **WHEN** 會話結束時生成總結
- **THEN** AI列出本次會話中表现最弱的2-3个方面
- **AND** 每个弱點附带具体例子（如"第3題回答時詞汇贫乏"）
- **AND** 提供針對性改進建議

#### Scenario: 跨會話持续弱點識別
- **WHEN** 係統分析最近10次會話的历史數據
- **THEN** AI識別持续出现的弱點（如"邏輯思維題平均分<60%"）
- **AND** 標注弱點的嚴重程度（轻微/中等/嚴重）
- **AND** 建議調整訓練計劃增加針對性練習

### Requirement: 參考答案生成
係統SHALL为每道題目生成高质量的參考答案。

#### Scenario: 生成標準參考答案
- **WHEN** 用户查看某題的反馈
- **THEN** 係統展示该題的參考答案（如果題庫中已有則直接显示）
- **AND** 如果没有，AI即時生成針對该題的優秀回答示例
- **AND** 標注參考答案的關键得分點

#### Scenario: 生成多樣化答案示例
- **WHEN** 用户请求查看多个參考答案
- **THEN** AI生成2-3个不同風格但都優秀的回答示例
- **AND** 注明每个示例的特點（如"简洁型"、"详细論证型"）

### Requirement: 學校針對性建議
係統SHALL根據目標學校的特點提供定制化建議。

#### Scenario: 針對SPCC的反馈
- **WHEN** 學生的目標學校是SPCC且完成練習
- **THEN** AI基于SPCC面試特點（如重视英文口語和国际视野）给出建議
- **AND** 指出回答中符合或不符合该校期望的部分
- **AND** 建議如何調整以更符合该校風格（如"SPCC重视批判性思維，可以在回答中加入不同觀點對比"）

#### Scenario: 針對QC的反馈
- **WHEN** 學生的目標學校是QC且完成練習
- **THEN** AI基于QC面試特點（如重视學術能力和邏輯思維）给出建議
- **AND** 强調學術相關的表现和改進點

### Requirement: 進步追踪反馈
係統SHALL在反馈中体现學生的進步情况。

#### Scenario: 對比历史表现
- **WHEN** 生成反馈時係統有历史數據可對比
- **THEN** 显示本次得分与历史平均分的對比（如"本次86分，比上周平均分78分提高了8分"）
- **AND** 指出進步的具体方面（如"詞汇使用明显丰富了"）
- **AND** 指出仍需改進的方面

#### Scenario: 里程碑成就反馈
- **WHEN** 學生某項指標達到预设目標（如"英文口語连续3次>85分"）
- **THEN** 係統给予正面鼓励反馈
- **AND** 建議提升该專項難度或增加其他專項練習

#### Scenario: 面試回憶反馈
- **WHEN** 用户提交真实面試回憶（包含問題和当時的回答）
- **THEN** AI分析当時表现的優缺點
- **AND** 指出回答中可以改進的地方
- **AND** 提供更好的回答示例
- **AND** 建議相似題型的練習方向

### Requirement: 反馈格式和展示
係統SHALL以清晰、易读的格式展示反馈內容。

#### Scenario: 結构化反馈展示
- **WHEN** 用户查看单題反馈
- **THEN** 按固定結构展示：語言质量、內容深度、具体建議、參考答案
- **AND** 使用評分、標籤、重點標注等方式增强可读性

#### Scenario: 會話總結报告
- **WHEN** 用户查看完整會話的總結
- **THEN** 展示：整体得分、各維度得分、弱點列表、進步對比、下次建議
- **AND** 支持導出为PDF或文本格式
